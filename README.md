## Documentation

http://docs.thesorachain.com

## Usage

### Build

```shell
$ forge build
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
forge script script/SoraContract.s.sol --broadcast --rpc-url $RPC_URL --private-key $PRIVATE_KEY
```
